1. Purpose of This Repository
The OPRMT™ Free Library provides structured prompt artifacts built using the OPRMT™ Framework.
This repository exists to:
Provide foundational prompt engineering assets
Demonstrate structured compliance methodology
Support educational use
Encourage disciplined prompt system design
This is NOT a general prompt dumping ground.
All contributions must preserve:
Structural integrity
Trademark integrity
Deterministic output logic
Framework compliance
2. Contribution Eligibility
Contributions are accepted if they:
Follow the OPRMT™ structure exactly
Improve clarity, consistency, or compliance
Add value within defined taxonomy categories
Do not violate licensing terms
Do not introduce proprietary content
The following will be rejected:
Creative-only prompts without structure
Rebranded OPRMT derivatives
AI-generated bulk uploads without validation
Sales funnels or commercial monetization overlays
Certification or scoring systems (enterprise only)
